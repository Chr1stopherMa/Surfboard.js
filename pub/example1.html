Making a Powerpoint with Surfboard
Press the Left and Right arrow keys to view the powerpoint
Step 1: Create the Slides
Each slide is an element with a width and height of 100% (filling the viewport). These elements are placed beside each other, essentially making a very wide page.


Using CSS Flexbox with a horizontal flow will help place the slides properly.
Step 2: Disable Scrolling
Set the overflow-x and overflow-y properties to 0 to prevent horizontal and vertical scrolling respectively.
Step 3: Fill in the Slides
Add content to your slides!

CSS Grid was used in this example to create a layout with 2 columns: One for the text and another for the image.
Step 4: Add the Code
Create the Surfboard object and call its scroll function. It's that simple!


'Right': the key pressed to go forwards

target: refers to the class name of the slides

offsetX: ensures the slide doesn't touch the leftmost part of the viewport. Since the slides in this example have a margin of 5% on all sides, we have a horizontal offset of -5%

reverse: the key pressed to go backwards
