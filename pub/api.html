Getting Started
Include the script tag above in the HTML file. You can instantiate a global Surfboard object by calling the Surfboard function. All methods below are properties of this object.
const surf = Surboard()
// go back when "B" is pressed
surf.redirect("B", "index.html")
Surfboard methods
pop(keys, element, options)
Pops (removes) the element from the screen.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
element
String or HTMLCanvasElement
The id of an element or the element itself.
options (optional)
Object
Customize the event options.
collapse:
if true, collapse the space when the element is popped; false by default
delay:
the transition time, in seconds, until the element fades out; 0 by default
toggle:
if true, the key event can be triggered multiple times to make the element disappear/re-appear; false by default
transition:
the transition timing function; "linear" by default
Returns
An object containing an enable and disable method (see below)
highlight(keys, element, options)
Visually highlights the element(s) by adding a border.

If multiple elements are specified, pressing the key multiple times will cause the highlighting to cycle through the elements. This can be used to let the user choose from a list of options (see return value).
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
element
String or HTMLCanvasElement or Array of HTMLCanvasElement
The id of an element, class of the element(s) or the element(s) itself.
options (optional)
Object
Customize the event options.
colour:
the colour of the highlight; no colour by default
reverse:
the key to be pressed to highlight the previous element; no key specified by default. This only has an effect if there are multiple elements
width:
the width of the highlighting border; "3px" by default
Returns
An object containing an enable and disable method (see below)

In addition, if the element argument was a class name or array of HTMLCanvasElement, the object also contains a selected() method which returns the HTMLCanvasElement that is currently highlighted.
expand(keys, element, options)
Expands the element by a specified factor.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
element
String or HTMLCanvasElement
The id of an element or the element itself.
options (optional)
Object
Customize the event options.
delay:
the time it takes for the image to expand in seconds; 1 by default
expand:
the horizontal and vertical scaling factor; 1 by default
expandX:
the horizontal scaling factor; 1 by default. Overrides the value in "expand" if specified.
expandX:
the vertical scaling factor; 1 by default. Overrides the value in "expand" if specified.
toggle:
if true, pressing the keys will toggle the element between its expanded and original sizes; otherwise, the element will return to its original size when the key event is finished; false by default.
Returns
An object containing an enable and disable method (see below)
slide(keys, element, options)
Translates the element by the specified distance.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
element
String or HTMLCanvasElement
The id of an element or the element itself.
options (optional)
Object
Customize the event options.
delay:
the time it takes for the image to expand in seconds; 1 by default
toggle:
if true, pressing the keys multiple times will cause the element to switch between its original and slided positions. Otherwise, the element will stay at its slided position once the event is triggered; false by default
translateX:
the amount in pixels to horizontally translate the element; 0 by default
translateY:
the amount in pixels to vertically translate the element; 0 by default
Returns
An object containing an enable and disable method (see below)
scroll(keys, options)
Moves the viewport to a specified location.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
options (optional)
Object
Customize the event options.
horizontal:
the amount of horizontal scrolling. If it is an integer, it represents the number of pixels. If it is a floating point number, it represents a percentage of the viewport width; 0 by default
offsetX:
the horizontal offset in pixels. Negative values represent an offset towards the left; 0 by default
offsetY:
the vertical offset in pxiels. Negative values represent an offset towards the top of the screen; 0 by default
relative:
if true, the scrolling occurs relative to the current position of the viewport; false by default
reverse:
if "target" is defined, this represents the key pressed to scroll in the reverse direction; no key specified by default
smooth:
if true, sets the scroll-behavior to smooth; false by default.
target:
the id or classname of an element(s) or the element(s) itself. For a single element, this will cause the screen to move to the top-left corner of the element. For multiple elements, triggering the key event multiple times will cause the screen to move through the elements
vertical:
the amount of vertical scrolling. If it is an integer, it represents the number of pixels. If it is a floating point number, it represents a percentage of the viewport width; 0 by default.
Returns
An object containing an enable and disable method (see below)
redirect(keys, link, callback)
Redirects the user to the specified link.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
link
String
The URL path. Can be relative or absolute.
callback (optional)
Function
A function to be called before the redirection occurs (allows for any "clean-up" required).
toggle (optional)
Boolean
If true, callback/callback2 will alternate whenever the key event occurs. Otherwise, callback2 will be called whenever the key event finishes; false by default
Returns
An object containing an enable and disable method (see below)
custom(keys, callback, callback2, toggle)
Triggers a user-defined event.
Argument
Type
Description
keys
String
The combination of keys to be pressed (see below).
callback
Function
The callback function to be called.
callback2 (optional)
Function
A second callback function. It's timing depends on the toggle argument below; an empty function by default
toggle (optional)
Boolean
If true, callback/callback2 will alternate whenever the key event occurs. Otherwise, callback2 will be called whenever the key event finishes; false by default
Returns
An object containing an enable and disable method (see below)
Key String Format
All methods take a string representing the key(s) to be pressed as their first parameter. The string should be a sequence of one or more keys separated by "+".

Note that order does not matter. For example, "Ctrl+J" is the same event as "J+Ctrl".

For methods that differentiate between pressing and holding, prefix the string with "hold+" to specify that the keys need to be held.

See below for a comprehensive list of all the keys supported.
Event String
When A is pressed "A"
When A and B are pressed "A+B"
When the Space bar is pressed "Space"
When Control and C are pressed "Ctrl+C"
When 1 and the Right arrow key are pressed "Right+1"
When K is being held "hold+K"
When M and N are being held "hold+M+N"
Key String Code
Alphanumeric "A-Z0-9"
Left Control "Ctrl"
Right Control "RCtrl"
Left Shift "Shift"
Right Shift "RShift"
Enter "Enter"
Space Bar "Space"
Left Arrow Key "Left"
Right Arrow Key "Right"
Up Arrow Key "Up"
Down Arrow Key "Down"
Period "."
Comma ","
Forward Slash "/"
Equal "="
Minus "-"
Left Bracket "["
Right Bracket "]"
Return Value
All methods return an object containing two functions:

disable()
Removes the key event listener associated with the function call that created this object. This method has no effect if the event listener has already been removed.
enable()
Adds the key event listener associated with the function called that created this object. This method has no effect if the event listener is already being listened for.
