Surfboard.js

"Surf the web using your keyboard"
Getting Started
Create a global object that keeps track of keyboard events
const surf = Surfboard()
Basic Functions
Pop Elements
Press "Q" to pop this element
Press "E" to pop with a transition
Press "T" to pop and collapse the space
Press "Y" multiple times to toggle
Combine multiple options (press "U" multiple times)
surf.pop('Q', "#pop-1")
surf.pop('E', "#pop-2", {delay: 1})
surf.pop('T', "#pop-3", {collapse: true})
surf.pop('Y', "#pop-4", {toggle: true})
surf.pop('U', '#pop-5', {toggle: true, delay: 0.5})
Slide Elements
Press "Z" to slide this to the left
Press "X" mulitple times to toggle
Press "C" to slide this up
Press "V" to slide down and to the right
surf.slide("Z", "#slide-1", {translateX: '-200%'})
surf.slide("X", "#slide-2", {translateX: '-200%',
    delay: 0.5, toggle: true})
surf.slide("C", "#slide-3", {translateY: '-220%'})
surf.slide("V", "#slide-4", {translateX: '150%',
    translateY: '300%'})
Highlighting Elements
Press "I" to highlight this element
Hold "O" to highlight this element white
Press "P" multiple times
to cycle through
these 3 elements
surf.highlight('I', "#highlight-1", {toggle: true})
surf.highlight('O', "#highlight-2", {colour: "#EEEEEE"})
surf.highlight('P', ".highlight")
Expand Elements
Hold "A" to expand image to double its size
Hold "D" to expand horizontally (with no delay)
Hold "F" to expand unequally in both dimensions
Toggle instead of hold (press "G" multiple times)
Hold "H" to shrink the image
surf.expand('A', '#img-1', {expand: 2})
surf.expand('D', '#img-2', {expandX: 2, delay: 0})
surf.expand('F', '#img-3', {expandX: 1.5, expandY: 3})
surf.expand('G', '#img-4', {expand: 2, toggle: true})
surf.expand('H', '#img-5', {expand: 0.5, delay: 0.5})
Scroll Events
Hold "W" to scroll up the page
Hold "S" to scroll down the page
Try comparing the result to using the arrow keys!
Press "J" to instantly go to the top of the page
Press "M" multiple times to cycle through all the headings
surf.scroll('W', {vertical: -0.08, relative: true,
    smooth: true})
surf.scroll('S', {vertical: 0.08, relative: true,
    smooth: true})
surf.scroll('J', {vertical: 0})
surf.scroll('M', {target: '.heading',
    offsetY: -30, smooth: true})
Custom Events
Not satisfied with the options above?
Pass in your own callback function to be called
Pressing "R" will refresh the page, un-doing some of the actions from this demo
let callback = () => window.location.reload()
surf.custom('R', callback)
More Examples
Toggle the Navigation Bar
Press "N" to toggle the nav bar
surf.slide('N', '#nav-bar', {toggle: true,
    translateX: '100%', delay: 0.25})
Remove Pop-Up Notifications
Press "." to simulate a pop-up notification
Press "X" to remove the pop-up
let notifcation = document.getElementById("notif")
function notify() {
    notifcation.removeAttribute("style");
    notifcation.style.visibility = 'initial';
}
surf.pop('X', '#notif')
surf.custom('.', notify)

Advanced Usage
Turning off Events
All functions return a handler that contains two methods
handler.disable disables the event
handler.enable enables the event
const handler = surf.pop('Q', "#pop-1")
handler.disable()
// event no longer occurs when "Q" is pressed
handler.enable()
// event is re-enabled
Binding Multiple Keys
Trigger events only when a combination of keys are pressed
Use "+" to separate keys
Press "Ctrl+Q"
Press "Ctrl+E"
surf.highlight('Ctrl+Q', '#combination-1',
    {toggle: true})
surf.highlight('Ctrl+E', '#combination-2',
    {toggle: true})
